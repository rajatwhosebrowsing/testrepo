D] Illega; Turns/Uturns:
What you need:
Vehicle trajectory
Allowed direction per lane (map / config)
Approach:
Track vehicle path using DeepSORT / ByteTrack
Compute angle change
Compare with allowed turn
IF turn_angle ∈ forbidden_range
→ violation

E]Illegal parking
What you need
Vehicle detection
Time-based logic
Approach
Define No-parking zones
Track stationary vehicle
Timer-based rule
IF speed ≈ 0 for > X seconds
AND location ∈ no-parking zone
→ violation


F] Wrong side driving:
Approach
Lane direction map (static)
Track vehicle motion vector
Compare with lane direction
IF dot(direction_vehicle, direction_lane) < 0
→ violation

G]Helmet/seatbelt:
No helmet / seatbelt
Approach
Object detection / pose-based
Detect:
person
helmet
seatbelt
Check spatial relationship
IF rider_detected AND helmet NOT detected
→ violation
Data:
Helmet detection datasets
Roboflow has good ones
